## What I learned

#### React Native

I started writing it while learning react native, it helped me a lot to build a real project to understand the fundamentals. <br />
I'm using [Expo](https://expo.io/), it comes with a nice developer experience and facilitates **a lot** of things.
However, in the futur, I might eject from **Expo managed workflow** to have more freedom over my app bundle, native libraries etc...

#### Typescript

As I decided to learn **Typescript**, I rebuilt **TOZ** using it. <br />
