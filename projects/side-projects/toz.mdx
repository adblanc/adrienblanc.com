## What is it ?

TOZ is my first react native project.<br />
It is availaible on both the _[App Store](https://apps.apple.com/fr/app/toz-jeux-dalcool-en-soir%C3%A9e/id1476978964#?platform=iphone)_ and _[Google Play](https://play.google.com/store/apps/details?id=com.blincio.toz&hl=fr&gl=US)_ and has more than **200K downloads**.<br />
I started writing it while learning react native, it helped me a lot to build a real project to learn react native fundamentals.
